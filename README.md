# 🚀 TaskFlow Kanban Project

A modern full-stack task management system inspired by tools like Trello and Notion.

## 🔥 Features

- 🔐 User Authentication (Signup & Login)
- 👑 Admin & User Roles
- 📋 Kanban Board (Drag & Drop)
- 📊 Task Analytics (Pie Chart)
- 🗑️ Admin Task Management (Delete tasks)
- 🎨 Modern Dashboard UI (TaskFlow style)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python (Flask)
- **Database:** PostgreSQL
- **Libraries:** bcrypt, psycopg2
